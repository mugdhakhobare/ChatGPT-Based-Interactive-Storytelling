
# 📖 ChatGPT-Based Interactive Storytelling

## 🚀 Project Overview

This project is an interactive storytelling system built using Python in a Jupyter Notebook. It simulates how AI-powered tools like ChatGPT generate dynamic narratives based on user input.

The user participates in the story by making decisions, and the system generates different story paths accordingly.

---

## 🎯 Objective

* Create an interactive storytelling experience
* Simulate Generative AI-based narrative generation
* Implement decision-based branching logic
* Enhance creativity using Python

---

## 🛠️ Tech Stack

* Python 🐍
* Jupyter Notebook 📓
* Core Concepts: Lists, Dictionaries, Functions, Loops

---

## ✨ Features

* 📌 Story theme and character creation
* 🎭 Interactive storytelling flow
* 🔀 Decision-based branching paths
* 💬 Dynamic story generation
* 🏁 Final story output

## 🧠 How It Works

* The story starts with predefined characters and theme
* The user is given multiple choices
* Based on input, different story paths are generated
* The process continues until a final conclusion is reached
